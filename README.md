# Question_Answer_Croagunk

## Environment Setup
```
pip3 install nltk
pip3 install stanza
```   

### Usage
```
./ask <article> <n>
./answer <article> <questions>
```

### Stanfordcorenlp usage example
`python3 corenlp.py`

## Article Analysis
- Using stanza: 
```
python3 processor/stanza_process.py <article>
```
- Using nltk:
```
python3 processor/nltk_process.py <article>
```
